# S-P-500-prediction-with-LSTM-and-XAI-IG-and-Occ-
A LSTM model is used to predict the prices of the S&P 500. These predictions are then explained using two XAI methods.

Copy this command into your terminal to install the needed libraries:
pip install yfinance pandas numpy scikit-learn torch captum matplotlib

Make sure the file is named main.py and run it with the following command: python main.py

If the code shows an error the first time you run it, do not worry. Just run the command python main.py again. It usually needs a second try to work.
    
The model will train for 100 epochs and then three graphs will pop up. Between these graphs and at the end, additional information will also be printed in the terminal.
